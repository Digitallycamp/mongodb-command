# mongodb-command
reference : https://www.mongodb.com/docs/mongodb-shell/run-commands/

# start mongosh
``bash
mongosh
```
hit enter
# To display the database you are using, type db:

```bash
db
```

#To list the databases available to the user, use the helper show dbs.

```bash
show dbs
```

# To switch databases,
```bash
use <database>
```
